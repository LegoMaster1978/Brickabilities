P.I.X.E.L.

Photo-to-Instruction eXport for Engineered LEGO Mosaics

A Brickabilities project.

Overview

P.I.X.E.L. is a software application designed to convert photographs into LEGO-compatible mosaic build plans.

The long-term goal is to allow users to upload an image and automatically generate:

Pixelated mosaic artwork
LEGO color matching
Piece counts by color
Material cost estimates
Printable build instructions
Why I Built This

As a LEGO mosaic artist and creator, I wanted a tool that could simplify the process of converting photographs into large-scale LEGO artwork.

This project also serves as a software development portfolio project while I pursue a degree in Computer Science and continue developing my programming skills.
